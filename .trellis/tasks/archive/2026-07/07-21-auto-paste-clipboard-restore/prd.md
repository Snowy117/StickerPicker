# 自动粘贴与剪贴板恢复

## Goal

让用户从快捷键唤出的 StickerPicker 中选择表情后，可以无缝回到原应用并粘贴，同时在不破坏用户后续复制内容的前提下，按设置恢复选择前的剪贴板。

## Background

- 当前选择表情会把图片写入剪贴板。
- 当前任意左键选择在复制成功后都会隐藏窗口；搜索内容不会随之清空。
- 当前全局热键仅切换 StickerPicker 的显隐状态，尚未保存唤起前的前台窗口。
- 当前 Windows 剪贴板实现会先清空剪贴板，再写入 `CF_HDROP` 与可用时的 `CF_DIB`；尚无快照、变更检测或恢复能力。
- 用户希望减少“选择表情后再手动切回并按 Ctrl+V”的操作。
- 剪贴板中的表情可能只是一次性中转内容，因此需要可配置的延迟恢复机制。

## Requirements

- 新增“自动粘贴”设置项。
  - 启用后，用户选择表情时，程序应结束本次选择交互、清空搜索、让焦点回到唤起前的窗口，并向其发送粘贴快捷键。
  - 用户选择表情时若按住 `Alt`，本次跳过自动粘贴。
  - `Alt` 只跳过发送粘贴快捷键；复制、清空搜索、按“点击后不关闭窗口”决定显隐、恢复原窗口焦点以及按设置启动剪贴板恢复仍照常执行。
  - 只有通过全局热键从其他应用唤起时，才捕获该应用作为本轮自动粘贴目标。
  - 通过托盘显示、托盘设置或程序首次启动进入时，不发送粘贴快捷键，也不沿用历史目标。
- 新增“剪贴板自动恢复读秒”设置项，用于配置选择表情后多少秒尝试恢复选择前的剪贴板。
  - 取值为 `0–60` 的整数秒，默认 `0`；`0` 表示关闭自动恢复。
  - 到期前若剪贴板已被其他操作改变，则不得覆盖新内容。
  - 到期时若剪贴板仍是本次选择写入的表情，则移除该表情并恢复此前内容。
  - 恢复采用保守的全有或全无策略：只有选择前的全部剪贴板格式都能可靠快照时，才启动本次恢复倒计时。
  - 空剪贴板属于可恢复状态，到期时恢复为空。
  - 若任一格式无法可靠快照，仍正常复制并按条件自动粘贴，但本次不启动恢复；状态栏提示已跳过恢复。
  - 单个剪贴板格式快照上限为 `64 MiB`，全部格式合计上限为 `128 MiB`；超过任一上限按无法可靠快照处理。
  - 倒计时结束、取消或因外部变化失效后，立即释放快照占用的资源。
  - 倒计时期间监听剪贴板变更；检测到本次操作之外的序列号后立即取消恢复链并隐藏倒计时，不等待原截止时间。
  - 到期时使用本次操作的专用剪贴板标记和写入后的剪贴板序列号双重确认；任一不匹配即取消，不修改剪贴板。
  - 连续选择时只保留一个活动恢复链：若剪贴板仍是上一次由 StickerPicker 写入的表情，则沿用首次选择前的快照、用最新表情替换待确认内容并重新开始完整倒计时。
  - 若连续选择前检测到剪贴板已被外部操作改变，则旧恢复链失效；以当前新内容重新尝试完整快照并建立新的恢复链。
- 新增“点击后不关闭窗口”设置项。
  - 此项与“自动粘贴”互斥，设置界面不能保留两者同时启用的有效状态。
  - 用户在界面开启其中一项时，立即关闭另一项，以最近选择为准。
  - 加载到两项同时启用的配置时，保留“点击后不关闭窗口”、关闭“自动粘贴”，并持久化归一化结果。
- 每次复制成功后均清空搜索；是否隐藏窗口由“点击后不关闭窗口”决定，与本轮是否存在自动粘贴目标无关。
- 主窗口状态栏右侧显示剪贴板恢复倒计时进度，能表达距离恢复还有多少时间。
- 设置对话框的标题区保持固定，内容区在可用高度不足时提供垂直滚动，确保所有设置在小窗口下仍可访问。
- 未启用新增功能时，现有复制表情行为保持兼容。

## Acceptance Criteria

- [ ] 启用自动粘贴并通过全局快捷键唤起后，普通选择会把表情粘贴到此前处于前台的应用。
- [ ] 上述场景中，选择完成后搜索框被清空，StickerPicker 不再占据前台。
- [ ] 从托盘或程序首次启动进入时不自动粘贴，也不会误用此前捕获的窗口；复制后的清空与关闭行为仍遵循设置。
- [ ] 按住 `Alt` 选择时，本次不发送粘贴快捷键。
- [ ] 按住 `Alt` 选择时，除不发送粘贴快捷键外，其余选择后行为与普通选择一致。
- [ ] 启用剪贴板恢复后，倒计时结束且剪贴板未改变时，能恢复选择前内容。
- [ ] 倒计时期间用户复制新内容后，程序不会覆盖该内容。
- [ ] 选择前剪贴板含有无法可靠快照的格式时，本次跳过恢复且不会用不完整快照覆盖原内容。
- [ ] 因不支持格式或 `64 MiB` 单格式 / `128 MiB` 总量上限跳过恢复时，状态栏显示跳过原因且不出现活动倒计时。
- [ ] 选择前剪贴板为空且倒计时期间未发生其他更改时，倒计时结束后剪贴板恢复为空。
- [ ] 连续选择多个表情且期间无外部剪贴板变化时，只保留最近一次倒计时，最终恢复首次选择前的内容而不是前一个表情。
- [ ] 外部剪贴板变化会在倒计时到期前使恢复链失效，及时释放快照并移除活动进度。
- [ ] 到期时专用标记不匹配或序列号不匹配均会独立取消恢复，且不会修改剪贴板。
- [ ] 状态栏右侧能显示并更新当前恢复倒计时进度；倒计时结束、取消或失效后不再显示活动进度。
- [ ] 设置内容超过可用高度时可垂直滚动，标题和关闭入口保持可见。
- [ ] “自动粘贴”和“点击后不关闭窗口”不能同时生效，配置加载、界面修改与持久化后都满足该约束。
- [ ] 关闭自动粘贴与自动恢复时，选择表情仍保持现有复制行为。
